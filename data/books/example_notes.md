# Example Notes

This file shows the format for your knowledge base.

You can place book notes, distilled ideas, checklists, frameworks, and SOPs in this folder.
Each `.md` or `.txt` file will be chunked and indexed.

## Example framework

When solving an operational problem:
1. Define the real bottleneck.
2. Find the highest-leverage fix.
3. Test the fix on a small scale.
4. Measure whether it reduced time, cost, or errors.
5. Standardize only after it works.
